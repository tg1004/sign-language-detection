# Sign Language Detection

This project implements a sign language detection system using:
- MediaPipe for hand landmark detection
- Machine Learning (landmark-based)
- CNN (upgrade phase)

## Features
- Real-time sign language detection
- Sentence construction
- Delete last letter / word
- Text-to-speech output
- CNN-ready dataset collection

## Dataset
This project uses self-made dataset

## How to Run
1. Create virtual environment
2. Install dependencies
3. Run data collection or training scripts

## Tech Stack
- Python
- OpenCV
- MediaPipe
- Scikit-learn
- TensorFlow (CNN upgrade)
